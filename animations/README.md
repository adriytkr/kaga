# Animations

This directory contains the code that powers all the animations you see in the videos and throughout the articles.

- [Folder Structure](#)

## 📁 Folder Structure

The animations are located in the [/src](./src) directory.

Each topic is organize into its own folder. Withing each topic folder, there are two subdirectories:

- /video — contains the code used to generate the animations for the Youtube Videos.
- /article — contains the code for creating the Presentation Slides.
